📊 FastAPI - Sistema de Gestión Escolar

Este proyecto consiste en una aplicación desarrollada con FastAPI para la gestión de datos en una institución educativa. La aplicación permite registrar, actualizar, consultar y eliminar información de:

  • 👩‍🎓 Estudiantes: nombre, grado, curso y especialidad
  • 👨‍🏫 Profesores: nombre y materia impartida
  • 👨‍💼 Empleados: nombre y departamento

Este sistema implementa operaciones CRUD (Create, Read, Update, Delete) conectadas a una base de datos mediante SQLAlchemy, además de renderizar páginas con Jinja2 y manejar estilos con archivos estáticos (HTML y CSS).

Herramientas utilizadas: FastAPI, SQLAlchemy, Jinja2, HTML, CSS.
